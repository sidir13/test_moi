## Implémentation (référence pour le code)

Pour aligner Tailwind / CSS sur ce design system, préférer des **variables sémantiques** (`--brand-blue-dark`, etc.) mappées aux valeurs ci-dessus plutôt que des couleurs arbitraires. Voir `docs/DESIGN_SYSTEM.md` comme source de vérité ; la règle Cursor `design-system.mdc` impose leur usage sur le frontend.
